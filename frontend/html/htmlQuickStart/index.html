a超链接

通常用a标签指向其他网页，常用的用法有三种。

  • 指向外网页面
  • 指向本网其他页面
  • 页内跳转

用法1案例：

·<a href="https://www.baidu.com/" target="_blank">百度首页</a>

其中href里面写你要跳往的网址，这是必选的，这里链向的是百度的首页面。而target是可选的，常见的设值是_blank,主要是用于新建一个窗口，并在新窗口中跳往href指向的页面。

用法2案例：

<a href="002img.html">002img.html</a>

这个和用法1没什么区别，就是跳往的页面是本网站的页面而已，采用的方式既可以是相对，也可以是绝对，和IMG图片是类似的。

插件2Prettier - Code formatter

这个主要是格式化工具，比如你的HTML代码缩进问题，大段连续空格问题等等。


插件3Bracket Pair Colorizer

保证一对一对的括号以不同颜色的形式出现，提高代码的可读性。


插件4Highlight Matching Tag

当你点击标签时，会使得成对的标签有效果。

关于这个的配置在该插件里面有提示，你可以将这些配置复制粘贴到settings.json里面。关于如何个性化配置，建议学了CSS的盒子模型之后再来调，现在采用默认配置即可。


插件5Indent Rainbow

该插件的效果是使得你的缩进有颜色体现，增加代码的可读性。
